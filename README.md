
# Superstore Sales Data Analysis

## Project Overview
This project analyzes a retail Superstore dataset to discover sales trends, profit patterns, and business insights using Python and data visualization.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Sales by Category
- Top Cities by Sales
- Discount vs Profit Relationship
- Profit by Sub-Category

## Key Insights
- Technology products generate the highest sales.
- New York City and Los Angeles have the highest total sales.
- Higher discounts often lead to lower profits and sometimes losses.
- Copiers and Phones are among the most profitable products.
- Tables and Bookcases generate negative profit.

## Conclusion
The analysis shows that product category, location, and discount strategies significantly affect profitability. Businesses should focus on high-performing products and carefully manage discount strategies.
